# efg-3d-electromagnetics-2006
Reprodução didática em C/C++ do artigo de Parreira et al. (2006) sobre o método Element-Free Galerkin aplicado a problemas eletromagnéticos tridimensionais, com tradução comentada, formulação matemática, testes numéricos e comparação com FEM.
